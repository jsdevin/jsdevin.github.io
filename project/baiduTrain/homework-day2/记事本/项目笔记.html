# 使用flex布局完成记事本

# 项目背景

使用flex布局完成下图的静态页面。已完成的线上版本open in new window

# 需求分析与实现

  1. 分析第一层布局

    div的高度是靠内容撑开的，如果只是对外面的第一层开启flex布局，而div里没有内容的话，在页面中是看不到东西的，所以多分析了一层，就是左侧的个人中心。

    在最外层的布局中，有两个红色的方框，div方框默认是换行的，为了使得两个div在同一行，就要开启flex布局，开启了flex布局就要考虑主轴的方向，主轴的方向默认是从左向右的，在这里刚好符合要求，不需要进行改变。 在左边的那个红色方框内，有两个黄色div方框，div默认是换行的，所以这里的两个div就不需要开启flex布局了。

  2. 分析左边的个人中心部分 只有一个需要开启flex布局，其它使用div默认的换行功能即可

  3. 分析右边的serf、记事本、学习建议部分

  4. 对记事本和学习建议这两块的分析

# 我做了什么

依据给出的静态图，逐层分析各个结构，有思路有条理地进行布局。同时运用flex的实现这个小项目。

# 我学到了什么

通过这次实践，我对flex语法越发熟悉，对html、css的使用越发熟练，让我对html、css产生信心，不再感觉自己html、css知识很薄弱，起码能完成基础的页面布局。 这是增强信心的一次实战！
